# 알림톡 관리 시스템 - 설치 및 사용 가이드

## 1. Python 설치 (처음 한 번만)

### Windows PC에 Python 설치하기

1. 아래 링크에서 Python을 다운로드합니다:
   - https://www.python.org/downloads/
   - **"Download Python 3.12.x"** 노란 버튼 클릭

2. 다운로드된 파일을 실행합니다.

3. **중요!** 설치 화면 하단의 **"Add python.exe to PATH"** 체크박스에 반드시 체크하세요.

4. **"Install Now"** 클릭하여 설치합니다.

---

## 2. 프로그램 실행 방법

### 방법 A: 더블클릭으로 실행 (가장 쉬움)

1. `alimtalk_manager.py` 파일을 바탕화면에 놓습니다.
2. 파일을 **더블클릭**하면 프로그램이 실행됩니다.

### 방법 B: 명령 프롬프트에서 실행

1. `Win + R` 키를 누르고 `cmd` 입력 후 Enter
2. 아래 명령어 입력:
```
cd 바탕화면
python alimtalk_manager.py
```

---

## 3. .exe 파일로 만들기 (선택사항)

Python 없이도 실행 가능한 .exe 파일을 만들려면:

1. 명령 프롬프트(cmd)를 열고 아래 명령어 입력:
```
pip install pyinstaller
```

2. `alimtalk_manager.py` 파일이 있는 폴더에서:
```
pyinstaller --onefile --windowed --name "알림톡관리" alimtalk_manager.py
```

3. `dist` 폴더 안에 `알림톡관리.exe` 파일이 생성됩니다.
4. 이 파일을 바탕화면에 놓고 더블클릭하면 바로 실행됩니다.

## 4. 프로그램 사용법

### 대시보드
- 프로그램을 열면 대시보드가 먼저 보입니다
- 총 고객 수, 오늘 차감 건수, 미발송 알림, 총 잔여 시간을 한눈에 확인

### 고객 등록
1. 왼쪽 메뉴에서 **"👥 고객 관리"** 클릭
2. **"＋ 고객 등록"** 버튼 클릭
3. 고객명, 전화번호, 게임명, 총 구매 시간 입력
4. **"저장"** 클릭

### 시간 차감 (핵심 기능)
1. 왼쪽 메뉴에서 **"⏱️ 시간 차감"** 클릭
2. 고객 선택 → 기사 선택 → 플레이 시간 입력
3. 오른쪽에 카카오톡 메시지 미리보기가 자동 표시됩니다
4. **"✈️ 차감 등록"** 버튼 클릭
5. 메시지가 자동으로 클립보드에 복사됩니다
6. 카카오톡을 열고 **Ctrl+V**로 붙여넣기 하세요

### 기사 관리
1. 왼쪽 메뉴에서 **"🔧 기사 관리"** 클릭
2. 기사 등록/수정/삭제 가능
3. 시간당 단가를 설정하면 정산 금액이 자동 계산됩니다

### 설정
1. 왼쪽 메뉴에서 **"⚙️ 설정"** 클릭
2. 업체명 변경 가능 (기본: "리니지 학교")
3. 메시지 형식 수정 가능

## 5. 데이터 저장 위치

모든 데이터는 아래 폴더에 자동 저장됩니다:
```
C:\Users\{사용자이름}\알림톡관리_데이터\
```

- `customers.json` - 고객 정보
- `records.json` - 차감 기록
- `drivers.json` - 기사 정보
- `settings.json` - 설정

**백업**: 위 폴더를 통째로 복사해두면 백업이 됩니다.

---

## 6. 문제 해결

### 프로그램이 안 열려요
- Python이 제대로 설치되었는지 확인하세요
- 명령 프롬프트에서 `python --version` 입력 시 버전이 표시되어야 합니다

### 글씨가 깨져요
- Windows 10 이상에서 정상 작동합니다
- 한글 폰트(맑은 고딕)가 설치되어 있어야 합니다

### 데이터가 사라졌어요
- `C:\Users\{사용자이름}\알림톡관리_데이터\` 폴더를 확인하세요
- JSON 파일이 있으면 데이터가 보존되어 있습니다
